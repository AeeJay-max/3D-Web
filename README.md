# 3D Web Experiment

This is a personal 3D web experiment built as part of my learning journey as a Software Engineering student at HIT. The project is purely frontend-focused and serves as a visual showcase for experimenting with interactive 3D elements on the web.

## Live Demo

You can check out the live project here: [View Live Demo](https://3-d-web-eight-beta.vercel.app/)

## About the Project

- The Spline 3D object used in this project was sourced from a YouTube video description.

## Features

- Interactive 3D object that follows the pointer on the webpage
- Modern UI design with gradient and shadow effects
- Fully frontend implementation using React (or specify framework if different)

- Interactive pointer-based 3D effects
- Frontend UI experimentation and hands-on practice
